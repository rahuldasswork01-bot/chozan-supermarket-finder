# 🛒 Chozan Supermarket — Product Finder

A smart barcode-based supermarket product locator that helps 
customers find any product instantly without asking for help!

---

## 🌟 Features

- 📊 Search by Barcode Number
- 🏷️ Search by Product Name
- 📦 Search by Category
- 📍 Shows exact Aisle, Shelf & Side
- 🗺️ Visual Store Map with highlighted location
- ⚙️ Admin Panel to manage products
- ➕ Add & Delete products in real time
- 📱 Mobile Friendly Design
- 🎨 Premium Black & White Aesthetic UI

---

## 🚀 How to Run Locally

1. Clone the repository:# chozan-supermarket-finder
Barcode based supermarket product locator for Chozan Supermarket
git clone https://github.com/rahuldasswork01-bot/chozan-supermarket-finder.git
2. Install dependencies:
pip install flask
3. Run the app:
python app.py
4. Open browser:
http://127.0.0.1:5000
